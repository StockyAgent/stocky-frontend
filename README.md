# Stocky v2: 나만의 AI 주식 애널리스트 🚀

Stocky v2는 최신 UI 트렌드(Glassmorphism, Soft UI)와 직관적인 UX를 결합하여 설계된 주식/포트폴리오 분석 모바일 웹 애플리케이션입니다.
본 프로젝트는 기존 `/app/demo` 버전을 대대적으로 개편하여 파스텔톤 그린(`#1cb863`), 둥글둥글한 컴포넌트, 그리고 즉각적이고 생동감 넘치는 반응형 애니메이션을 도입했습니다.

## 🎯 주요 기능 및 아키텍처 (v2)

### 1. 사용자 흐름 (온보딩 및 페이지 구성)
- **온보딩 (`/v2/onboarding`)**: 직관적인 워딩과 함께 구글 소셜 로그인 인터페이스 제공.
- **코치 선택 (`/v2/coach`)**: 사용자의 성향에 맞는 투자 코치(테크, 매크로, 퀀트 등) 선택 및 유기적인 전환 애니메이션 제공. 코치 선택을 완료하면 `/v2/coach/confirm` 페이지에서 확인하거나, 마이페이지에서 접근한 경우 바로 이전 페이지로 복귀합니다.
- **관심 종목 설정 (`/v2/stocks`)**: 사용자가 초기에 관심 있는 주식을 선택할 수 있으며 다중 필터링 뱃지를 제공합니다.
- **홈 대시보드 (`/v2/home`)**: 시맨틱 태그 구조(`<main>`, `<section>`)로 작성되었으며, 오늘의 코치 브리핑, 실적 서프라이즈 등의 주요 위젯과 애니메이션(Slide-in, 카드 축소/확장) 중심의 대시보드 위젯 제공. 
- **관심종목 (`/v2/watchlist`) & 종목 편집 (`/v2/watchlist/edit`)**: 사용자의 현재 포트폴리오 평가 금액과 뱃지로 요약된 보유 주식 리스트 확인 및 편집 가능.
- **마이페이지 (`/v2/my`)**: 프로필 관리, 알림 설정, 다크모드/라이트모드 글로벌 테마 스위치 포함. 바텀시트 모달을 통해 인라인 프로필 픽처(이모지), 닉네임, 이메일 수정을 지원.
- **서비스 및 공지 (`/v2/terms`, `/v2/notices`)**: 약관 및 공지사항 상세 조회 지원.

### 2. 컴포넌트화 및 디자인 시스템
- **BottomTabBar (`/components/v2/BottomTabBar.tsx`)**: 홈, 관심종목, 마이페이지 등 주요 루트 페이지에 공통 삽입되어 코드 중복을 제거한 하단 네비게이션입니다. 선택(Active)시 인터랙션 애니메이션이 탑재되어 있습니다.
- **컬러 컨벤션**: Base 백그라운드는 옅은 민트 (`#f5faf7`), 주요 액센트 컬러는 Stocky Green (`#1cb863`), 다크모드는 `#121212` 및 `#1e1e1e` 기반의 세련된 차콜 그레이 톤으로 구현되었습니다. 

### 3. 데이터 및 아키텍처 리팩토링 포인트
- **시맨틱 HTML 적용**: `<div className="flex...">`로만 중첩되던 컴포넌트들을 `<main>`, `<header>`, `<section>`, `<nav>` 태그로 변환하여 접근성(a11y)과 유지보수성을 극대화했습니다.
- **데이터 모듈화 (`/data/v2/mock.ts`)**: 개발 단계 및 UI 시연에 사용되는 모든 더미 데이터(STOCKS, WATCHLIST, NOTICES 등)를 중앙 컨트롤 파일 1개로 병합하여 일관성을 높였습니다. 
- **Next.js & Tailwind CSS**: 서버/클라이언트 라우팅 최적화를 위해 Next.js의 App 라우터를 사용하고 있으며, 복잡한 인라인 css와 다크모드 적용 로직을 템플릿 리터럴과 Tailwind 클래스로 부드럽게 통합했습니다.

## 💻 시작하기

본 프로젝트는 Node.js 와 npm (또는 pnpm, yarn) 이 필요합니다.

```bash
# 1. 의존성 설치
npm install

# 2. 로컬 개발 서버 실행
npm run dev
```

서버가 켜지면 브라우저 상에서 `http://localhost:3000/v2/home` 혹은 `http://localhost:3000/v2/onboarding`을 통해 Stocky v2를 경험해보실 수 있습니다.
